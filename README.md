# EX-2(A)  PRINTING A STRING

## AIM:
To write a C Program to print the string "WELCOME" n number of times.

## ALGORITHM:
1. Start the program.
2. Read a variable.
3. Using for loop print the string n number of times.
4. Stop the program.

## PROGRAM:
```
#include<stdio.h>
int main()
{
   int n,i;
   scanf("%d",&n);
   for(i=1;i<=n;i++)
   {
       printf("WELCOME\n");
   }
   return 0;
}
```
